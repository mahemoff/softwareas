<a href="http://weborandom.com">Web-O-Random</a> is a new site I made, a simple random number generator with simple URLs. <a href="http://weborandom.com">The About Page</a> tells the story:

Random Numbers

* <a href="http://weborandom.com">http://weborandom.com</a> A number between 1 and 100
* <a href="http://weborandom.com/6">http://weborandom.com/6</a> A number between 1 and 6
* <a href="http://weborandom.com/-10/10">http://weborandom.com/-10/10</a> A number between -10 and 10
* <a href="http://weborandom.com/5/10.000">http://weborandom.com/5/10.000</a> A decimal number between 5 and 10, at 3 decimal place precision

Random Lists

* <a href="http://weborandom.com/larry,moe,curly">http://weborandom.com/larry,moe,curly</a> A random item from the list
* <a href="http://weborandom.com/larry,moe,curly/2">http://weborandom.com/larry,moe,curly/2</a> 2 items from the list, randomly selected and ordered
* <a href="http://weborandom.com/larry,moe,curly/0">http://weborandom.com/larry,moe,curly/0</a> The list, shuffled

Programming Formats

* <a href="http://raw.weborandom.com/10">http://raw.weborandom.com/10</a> Just the number, no formatting. Use it as an API.
* <a href="http://weborandom.com/10">http://weborandom.com/10</a> The number and parameters in JSON format

(<a href="http://www.youtube.com/watch?v=bZ9WhpQhv8g">The old Web-O-Random is late.</a> Deceased. Run down the curtain and joined the bleedin' choir. This is another of those things I was planning to do for several years and finally got around to it.)
